# 自動販売機シミュレータ　README.md

---

## 1. アプリの概要
このアプリは、ブラウザ上で動作する自動販売機シミュレータです。  
商品一覧の表示、金額の投入、商品の購入、返金処理など、実際の自動販売機の基本的な動作を再現しています。

---

## 2. 機能一覧
- 温かい／冷たい商品の分類表示（HOT / COLD）
- 商品一覧の表示（名前・価格・温度・在庫）
- 購入履歴等のログ表示
- 投入金額の管理（10円〜1000円まで）
- 在庫管理（購入ごとに減少、0で売り切れ表示）
- 商品の購入機能
- 金額不足時のエラーメッセージ表示
- 返金機能（投入金額をリセット）

---

## 3. 動かし方
- Node.jsがインストール済み

1. DevContainerでUsers/user/Projects/vending-machineを開く

2. ターミナルでサーバーを起動
node server.js

3. ブラウザで以下のURLにアクセス
http://localhost:3000
